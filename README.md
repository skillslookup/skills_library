# 🚀 Skills Lookup: Your AI Learning Hub

Welcome to **SkillsLookup**! This repository is designed to be the simplest, most educational, and structured way to start your journey into Artificial Intelligence.

Whether you are a complete beginner or a developer looking to dive into advanced concepts like Agents and RAG, this hub is for you.

---

## 🗺️ How to Use This Repo

Don't be overwhelmed! Follow our progressive learning path. Each folder contains a simple explanation, real-world context, and runnable code examples.

### 🏁 Suggested Learning Path
1.  **[Basics](./basics)** → Start here to understand what AI is and how it works.
2.  **[Intermediate](./intermediate)** → Learn how to give AI memory and search capabilities.
3.  **[Advanced](./advanced)** → Build autonomous agents and customize brains.
4.  **[Tools](./tools)** → Master the industry-standard libraries.
5.  **[Projects](./projects)** → Put it all together by building real apps.
